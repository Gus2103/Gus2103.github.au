About this project

Another uni project is this stool, it was also constructed during the TECH205 unit. This project was kind've inspired by my valet stand I made during 12, part of the project was that there were 3 levels, separated by thing strips of timber to make it look like it was floating, which is also what I've tried to do here for my stool. As for the pattern of the cushion, this was inspired by the playstation logo, this is because the intial design solution that the stool was meant to be fixing was that I wanted somewhere to put my feet up while playing, so I thought that it would be cool to theme it to it's use so that everyone knew what it was for.

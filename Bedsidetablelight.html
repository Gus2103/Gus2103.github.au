About this project

This project was another uni project, part of the TECH104 unit. As I mentioned in the About Me page I love footy and my favourite team is the South Sydney Rabbitohs which as you can see is a massive part of my design. The task brief for this project was to create a table lamp used for a nightlight/desk, this reminded me of a project i amade in year 7 or 8 while at school which is this exact idea, however I changed up the light up design, and gained an easy to use LED strip which sits inside the timber which I chose for this project, this project was a lot of fun to make and easy to do. One really cool thing about the LED strip is that it can change colours to whatever you want, or even make so it changes colours automatically.
